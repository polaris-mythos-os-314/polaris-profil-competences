# Instructions for Setting Up and Running Configuration Scripts

## English Instructions
1. **Prerequisites:** Make sure you have the following installed:
   - Python 3.x
   - Required libraries (can be installed via `pip install -r requirements.txt`)

2. **Setting Up the Environment:**  
   - Clone the repository: 
     ```bash
     git clone https://github.com/yourusername/polaris-profil-competences.git
     ```  
   - Navigate to the project directory:
     ```bash
     cd polaris-profil-competences
     ```

3. **Running the Scripts:**  
   - To run the configuration script, use:
     ```bash
     python config_script.py
     ```
